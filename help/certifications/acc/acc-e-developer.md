# Percorso di certificazione per Adobe [!DNL Campaign Classic] Sviluppatore esperto

Scopri come diventare un esperto Adobe Developer certificato.

## Esperienza minima

Dovresti avere:

* Almeno due anni di esperienza software Campaign
* Almeno tre anni di esperienza congiunta nella programmazione JavaScript, nello sviluppo SQL, incluso ETL, e nelle tecnologie web (XML, HTML e servizi web)
* Esperienza durante l’esecuzione di una campagna di marketing utilizzando un omnicanale digitale
* Esperienza con l&#39;architettura del database e conoscenza del database marketing
* Possibilità di mappare i requisiti alla funzionalità di Campaign
* Risoluzione dei problemi relativi alle implementazioni di Campaign
* Informazioni di base sulle configurazioni del server relative a Campaign

>[!NOTE]
>
>La tua certificazione è valida per due anni dal momento del passaggio. È consigliabile adottare la versione più recente quando possibile, in quanto copre le funzionalità di prodotto più recenti.

**Obiettivi e campo di applicazione dell&#39;esame**

Sezione 1: Configurazione del sistema (16%)

* Applica impostazioni di configurazione e impostazioni predefinite a livello di sistema
* Dati vari parametri, seleziona la funzione Adobe Campaign appropriata per implementare un caso d’uso
* Risoluzione dei problemi relativi alle impostazioni di configurazione del sistema
* Implementare configurazioni personalizzate
* Identificare le funzionalità di configurazione del Pannello di controllo Campaign Campaign

Sezione 2: Personalizzazione (22%)

* Dati un insieme di requisiti, applica le best practice per progettare un modello dati
* Descrizione dell’implementazione delle API in Campaign
* Descrizione delle best practice per lo sviluppo con JavaScript in Adobe Campaign
* Applicare librerie JavaScript nei flussi di lavoro, nelle chiamate SOAP, nelle regole di tipologia e nei blocchi di personalizzazione
* Descrizione delle best practice per l&#39;utilizzo delle funzioni SQL in Adobe [!DNL Campaign Classic]
* In uno scenario, comprendere come utilizzare gli script SQL in Adobe [!DNL Campaign Classic]
* Descrizione delle funzioni OOTB comunemente utilizzate

Sezione 3: Gestione degli accessi (8%)

* In base a uno scenario, configura le impostazioni di protezione per un operatore
* Descrivi come configurare le impostazioni di gestione degli accessi in base al documento di progettazione della soluzione
* Descrizione della gestione degli accessi tramite il pannello di controllo Campaign
* Identificare le limitazioni del partizionamento dei dati e quando utilizzare sysfilter

Sezione 4: Integrazione dei dati (16%)

* In uno scenario, seleziona l’approccio migliore per l’integrazione con origini dati esterne
* Implementare le best practice per l’elaborazione dei dati nei flussi di lavoro
* Applicare tecniche di integrazione dei dati all’interno di un flusso di lavoro
* Identificare le impostazioni di sicurezza e rete da integrare con Adobe Campaign da un sistema esterno
* Imposta il processo CRUD su uno schema dall&#39;esterno di Adobe Campaign

Sezione 5: Manutenzione delle istanze (12%)

* Comprendere ed agire sulla mappa di calore del flusso di lavoro
* Dimostrare il processo per creare un flusso di lavoro personalizzato o un rapporto per il monitoraggio delle istanze
* Identificare i problemi basati su indicatori di sistema
* Spiegare l&#39;impatto dei servizi e dei processi sul sistema
* Dimostrare metriche/controlli di audit del sistema di base

Sezione 6: Configurazione della campagna (25%)

* Applicare tecniche di personalizzazione avanzate
* Configurare e applicare mappature di destinazione personalizzate
* Identifica l’utilizzo JavaScript appropriato nel flusso di lavoro Campaign e perché
* Applica il codice personalizzato per manipolare le tabelle temporanee e il contesto del flusso di lavoro
* Convalida e debug della logica del flusso di lavoro
* Applicare impostazioni di consegna avanzate
* Descrivere le pratiche del flusso di lavoro che hanno un impatto negativo sulle prestazioni
* In base a un caso d’uso, consigliamo una progettazione complessa del flusso di lavoro di targeting

**Fai un test pratico**

Prima di iniziare, consigliamo di effettuare il test gratuito. Avrai accesso illimitato a tutti i test di esercitazione disponibili per gli esami a livello di esperti di Adobe.

I nostri test sono tutti sviluppati partendo dallo stesso modello degli esami in diretta. Il test di esercitazione ti aiuterà a comprendere gli argomenti trattati nell&#39;esame in diretta.

Ricordare che il superamento di un test di pratica non garantisce il superamento dell&#39;esame di certificazione.

Vai a [Portale di preparazione all&#39;esame di certificazione di Adobe](https://www.certmetrics.com/adobe/candidate/gmetrix_sso.aspx){target="_blank"} ora per fare un test di allenamento.

**Prendere l&#39;esame di certificazione**

Adobe utilizza due fornitori per amministrare gli esami elaborati, PSI ed Examity.

* **PSI** offre opzioni di test center e di esame giapponese (ove disponibili)
* **[!DNL Examity]** offre un&#39;opzione online remota

Per pianificare, riprogrammare o annullare l&#39;esame, è sufficiente fare clic sul fornitore desiderato e seguire le istruzioni fornite sullo schermo. È così semplice!
